# ToDoApp

## Project Description
ToDoApp is a simple yet powerful task management application that helps users manage their daily tasks efficiently. The app offers a user-friendly interface and essential features to add, edit, delete, and organize tasks into different categories. Designed to help improve productivity, ToDoApp is suitable for both personal and professional use.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/DeividMomchilov/ToDoApp.git
   cd ToDoApp
   ```

2. **Install dependencies**:
   Make sure you have Node.js and npm installed. Then run:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```
   Open your browser and go to `http://localhost:3000` to start using ToDoApp.

## Features
- Add, edit, and delete tasks
- Organize tasks into categories
- Mark tasks as completed
- Filter and search tasks
- User authentication (if applicable)
- Responsive design for mobile and desktop usage
