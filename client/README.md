# React ToDo App

A responsive and feature-rich ToDo List application built with React and Vite. 

## Features

* **Task Management**: Add, edit, and delete tasks easily.
* **Reorder Tasks**: Move tasks up or down to prioritize them.
* **Dual Themes**: Toggle between beautifully styled Light and Dark modes.
* **Bilingual Support**: Built-in localization for English (EN) and Bulgarian (БГ).
* **Responsive Design**: Optimized for desktop, tablet, and mobile viewing.

## Technologies Used

* **React** (v19)
* **Vite** (Build tool)
* **CSS3** (Custom styling with CSS variables and gradients)
